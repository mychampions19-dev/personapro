# Persona Pro — AI Avatar Prompt Engine

A powerful AI character creation tool that generates ultra-detailed prompts for AI image generators.

## Setup

1. Go to [openrouter.ai/keys](https://openrouter.ai/keys) and create a free API key
2. Open the app and paste your key when prompted

## Features

- 🔒 Identity Lock — 36 identity fields
- 📷 360° Shot Pack
- 🎭 Expression Sheet  
- 🗂️ Content Scene Pack (27 scenes)
- 🔍 Scan Image — auto-fill from any photo
- 🎬 Video Prompts (Kling, Veo 3, Runway)
